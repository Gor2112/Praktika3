Вариант 6. Приближенный алгоритм для TSP (2-аппроксимация)

Задача: реализовать простой 2-аппроксимационный алгоритм для TSP: построить минимальное остовное дерево, удвоить ребра, найти эйлеров обход и исключить повторения.

Требования:

· Входные данные: граф с весами ребер (полный граф, метрическое пространство)
· Выход: приближенный маршрут и его стоимость
· Вывести соотношение найденного решения к нижней оценке

Входные данные:
Матрица расстояний между 5 городами (полная матрица 5х5)
[0, 10, 15, 20, 25]
[10, 0, 35, 25, 30]
[15, 35, 0, 30, 40]
[20, 25, 30, 0, 45]
[25, 30, 40, 45, 0]

Краткий анализ алгоритма:

1. Ключевая логика (построение MST):
   · Алгоритм Прима для построения минимального остовного дерева
   · Начинаем с вершины 0, на каждом шаге добавляем кратчайшее ребро к новой вершине
2. Эйлеров цикл (строки 33-50):
   · Удваиваем все ребра MST для получения эйлерова графа
   · Находим эйлеров обход с помощью DFS
   · Сохраняем порядок посещения вершин
3. Построение гамильтонова цикла (строки 53-61):
   · Проходим по эйлерову пути и удаляем повторяющиеся вершины
   · Оставляем только первое вхождение каждой вершины
   · Замыкаем цикл - возвращаемся в начальную вершину
4. Результат:
   · Приближенный маршрут: [0, 1, 3, 2, 4, 0]
   · Стоимость маршрута: 160
   · Нижняя оценка (стоимость MST): 85
   · Соотношение решение/нижняя_оценка: 1.882
   · Эффективность решения: 160/85 ≈ 1.882

Временная сложность: O(n²)
Почему: Построение MST алгоритмом Прима доминирует O(n²), остальные шаги - O(n)

Быстрее полного перебора O(n!): Для n=5 полный перебор проверяет 120 маршрутов, для n=10 - уже 3.6 миллиона

Гарантия качества: Алгоритм гарантирует решение не более чем в 2 раза хуже оптимального для метрического TSP

Принцип работы:

1. MST дает нижнюю оценку - оптимальный маршрут TSP не может быть короче стоимости MST
2. Эйлеров цикл в удвоенном MST имеет стоимость 2×MST
3. Удаление повторов не увеличивает стоимость (благодаря неравенству треугольника)
4. Итоговая стоимость ≤ 2 × оптимальная_стоимость
